# AI Collaboration in Graphite

Graphite is a modern STEM diagramming tool developed using a **Human-Architect, Multi-AI-Agent** model.

## Collaboration Model

- **Human Architect**: Jacques Ho
  Responsible for product vision, architectural decisions, security, legal compliance, and final code review.
- **AI Agents**:  
  High-quality code, unit tests, and documentation are generated through a collaborative network of AI agents, including:
  - **Gemini 3**: Core architecture and UI integration.
  - **Claude 4.5**: STEM templates and testing frameworks.
  - **Codex/GPT 5.5**: Environment optimization and repository hygiene.

## Transparency Statement

We believe in the future of AI-native software development. While the majority of the code in this repository is AI-generated, every line has been reviewed and verified by a human maintainer to ensure it meets our engineering standards and pedagogical accuracy for STEM education.
